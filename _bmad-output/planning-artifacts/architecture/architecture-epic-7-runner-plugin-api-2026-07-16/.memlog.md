Prevents: runner-specific logic leaking outside one plugin module; incompatible per-runner reimplementations of selection/coverage/confidence. - (decision) AD-14 Multi-Suite Registry Model. Binds: registry, CLI register. Prevents: two suites of one project silently colliding on one projectId/config/coverage-map; ambiguous which-runner-for-this-project when >1 config exists. Rule: RegisteredProject gains 'suites: Record<suiteName, {configPath, plugin: RunnerPlugin['name']}>' alongside its existing projectId/path/status (additive, not a replacement). Everything currently keyed by projectId alone (coverage map, history, plans) additionally keys by suiteName, one directory level deeper under the existing per-project .test-mcp/ state topology (AD-8) - e.g. .test-mcp/<suite>/coverage-map.json. test-mcp register auto-detects suites via each installed plugin's detect(); an explicit --suite name:plugin:configPath override exists for when auto-detect fails or picks wrong - auto-detect is the default path, not the only one.
- (decision) AD-15 Per-Suite Selection/Coverage/Confidence Scoping. Binds: selection, coverage, combined coverage, confidence. Prevents: one suite's changed-file/coverage data being attributed to a different suite's tests (false confidence, wrong selection). Rule: SelectionEngine.plan, CoverageMapFile, and CombinedCoverage all become suite-scoped (add a suiteName dimension alongside the existing projectId), never merged across suites within one project. Confidence's existing {level:'high'|'degraded',reasons} union gains a third level 'unavailable', asserted whenever the suite's bound plugin declares capabilities.coverage==='none'. thresholdsMet stays undefined in that case, using the SAME gating pattern already proven at combined.ts:269-271 (today: thresholdsMet only set at confidence high) - never silently defaulted to a number or omitted without a reason.

Its test suite must pass the same hermetic-fixture-project pattern as the Vitest plugin's tests (proving the interface holds), not match its feature surface.
