Feature Categories

Test Priority Engine

  • Prioritize by git recency, failure history, business impact
  • Dynamic priority adjustment based on time/context
  • Auto-skip low-priority tests when constrained
  • Priority view showing top failures first

Delta Testing

  • Git-aware delta: only tests touching changed commits
  • AST-level delta: detect behavior-affecting changes
  • "Dry run" mode showing what WOULD run
  • Fallback to full suite when delta fails

Test Health Dashboard

  • Reliability score: pass rate over time
  • Speed score: p50/p95/p99 duration
  • Flakiness detector: intermittent failures
  • Health trend charts

Smart Retry Logic

  • Retry isolation: fresh process per attempt
  • Backoff strategy: exponential with jitter
  • Retry budget controls
  • Automatic flaky test flagging

Dependency Graph

  • Build-time graph generation
  • Circular dependency detection
  • Impact analysis: show what breaks on change
  • Graph export in DOT format

Progressive Disclosure

  • Level 1: Pass/fail counts + top 3 failures
  • Level 2: Add stack traces
  • Level 3: Add full stdout/stderr
  • Level 4: Raw JSON access

Cost Analysis

  • Per-test timing: min/avg/max/p95/p99
  • Cost ranking: top slowest tests
  • Parallelization efficiency metrics
  • Cost prediction before execution

CI-First Mode

  • Pre-flight checks: git status, disk space
  • Skip decision engine
  • Shard optimization
  • Artifact upload hints
